<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
```sh
git clone https://github.com/shahriya1995/amazon_clone_reactjs.git
```
2. Install NPM packages of front-end
```sh
cd amazon-clone
npm install
```
3. Install NPM packages of back-end
```sh
cd functions
npm install
```
